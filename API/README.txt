README: API Setup
1. edit mySQLconnect.js to reflect the users credentials
2. run npm install to download packages
3. run node api_server.js to start the api
4. test the connection with the url http://localhost:8443/api/v1/
5. login to a user in the database with http://localhost:8443/api/v1/login/admin/password/
6. see all tickets with http://localhost:8443/api/v1/ticket/all-tickets
